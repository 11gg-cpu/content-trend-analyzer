# Tagging Framework

## Core Tags

- Topic: career, AI tools, gaming, study, lifestyle.
- Format: tutorial, ranking, story, comparison, reaction.
- Hook: question, surprise, pain point, result-first, opinion.
- Audience: student, job seeker, gamer, creator, analyst.

## Review Questions

- What promise does the first three seconds make?
- Which audience problem does the video solve?
- Is the engagement driven by utility, identity, emotion, or controversy?
