# Trend Report

## Sample Insight

Ranking-style and tutorial-style videos show stronger engagement in the sample dataset because they make the viewer promise clear in the first few seconds.

## Metrics Used

- Engagement rate: `(likes + comments + shares) / views`
- Completion proxy: `average watch seconds / video seconds`
- Tag categories: topic, format, hook, audience

## Recommendation

Test content ideas as small experiments. Each experiment should have one main hypothesis, one audience, and one metric that decides whether the format is worth repeating.
